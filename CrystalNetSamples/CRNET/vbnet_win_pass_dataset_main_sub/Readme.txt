PRODUCT VERSION

This application is for use with Visual Basic .NET and Crystal Reports for Visual Studio .NET.

________________________________________________________________ 

DESCRIPTION

This Visual Basic .NET sample Windows application demonstrates how to pass a dataset to the main report and the subreport.

INSTALLATION

1. Extract the file 'vbnet_win_passDatasetToMainAndSubreport.exe' to the default installation folder, C:\Crystal\crnet\vbnet_win_passDatasetToMainAndSubreport\. 

2. Open the Visual Studio .NET IDE.

3. From the 'File' menu, click 'Open'| 'Project'. Browse to the location of the extracted sample application and select 'vbnet_win_passDatasetToMainAndSubreport.sln'. Click 'Open' to load the project.

4. From the 'Build' menu, click 'Build Solution' to compile the application.

5. From the 'Debug' menu, click 'Start Without Debugging' to run the application.

A report now appears in the viewer.
